Pending Transactions (Mobile)

How should pending transactions show on mobile?

A

Stacked Cards with Actions

Each pending tx is a card with merchant, amount, date. Approve/Dismiss buttons are large and full-width — easy to tap. Swipe-to-dismiss as bonus.

┌─────────────────────────┐
│ ⏳ 19 Apr    Gojek      │
│           Rp 25.000     │
│ [Approve ✓] [Dismiss ✗] │
├─────────────────────────┤
│ ⏳ 18 Apr    Grab       │
│           Rp 45.000     │
│ [Approve ✓] [Dismiss ✗] │
└─────────────────────────┘
B

Swipe Actions (Recommended)

Full-width card with merchant/amount/date. Swipe right to approve (green), swipe left to dismiss (red). Large tap targets, modern feel. Requires tap as fallback.

← Swipe │ Swipe →
┌─────────────────────────┐
│ ⏳ 19 Apr    Gojek      │
│           Rp 25.000     │
│           [tap: actions] │
└─────────────────────────┘
(GREEN = Approve, RED = Dismiss)

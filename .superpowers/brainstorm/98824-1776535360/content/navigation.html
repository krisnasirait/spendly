Navigation & Dashboard (Mobile)

How should the dashboard header work on mobile?

A

Stacked Header

Greeting takes full width, period selector + scan button stack below it. More vertical space but everything visible.

┌─────────────────────────┐
│ Good morning, John      │
│ 19 Apr 2026             │
├─────────────────────────┤
│ [Today][Week][Month]    │
│ [    Scan Emails    ]   │
├─────────────────────────┤
│ Stats cards (2x2 grid)  │
│ ...                     │
└─────────────────────────┘
B

Compact Header

Greeting smaller, period selector collapses to dropdown, scan button stays. Fits everything in one row. More space for content.

┌─────────────────────────┐
│ ☀️ Good morning, John   │
│ [Dropdown: This Month][📧]│
├─────────────────────────┤
│ Stats cards (2x2 grid)  │
│ ...                     │
└─────────────────────────┘
